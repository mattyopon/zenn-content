## TL;DR

家族のタスク管理をLINEグループで完結させるAIアシスタントBotを作りました。Claude API（claude-sonnet-4-20250514）を搭載し、自然言語でタスクの追加・確認・完了ができます。

## 背景

家族でのタスク管理って意外と難しい。Todoistやノーションを導入しても「アプリを開くのが面倒」で結局使われなくなる。でも **LINE は毎日開く**。

「LINEに話しかけるだけでタスク管理できたら最強では？」という発想で作りました。

## 機能

### タスク管理
- **追加**: 「明日までに牛乳買って」→ タスク登録 + 期限設定
- **一覧**: 「今のタスク教えて」→ 未完了タスク一覧
- **完了**: 「牛乳買った」→ タスク完了
- **削除**: 「牛乳のタスク消して」→ タスク削除
- **担当割り当て**: 「パパに掃除お願い」→ 担当者設定

### AI チャット
- タスク管理以外の会話にも対応
- **20メッセージのコンテキスト記憶** - 文脈を理解した応答
- グループチャット対応 - 家族全員で使える

## 技術スタック

```
LINE Messaging API ──> Express Server ──> Claude API
       │                     │           (claude-sonnet-4-20250514)
       │                     └──> SQLite (タスク・会話履歴)
       └──> @line/bot-sdk v9
```

- **Node.js + TypeScript**
- **@line/bot-sdk v9** - LINE Messaging API
- **Claude API** - 自然言語理解 + タスク抽出
- **SQLite** - ローカルDB（タスク、会話履歴）
- **Express** - Webhook サーバー

## Claude API の活用

ポイントは、Claude にタスク管理の「意図」を理解させること。System Prompt で以下を指示しています：

1. ユーザーの発言からタスク操作の意図を抽出
2. タスク操作（追加/完了/削除/一覧）の場合はJSON形式で返す
3. 雑談の場合は通常の会話として応答
4. 期限の曖昧な表現（「明日」「来週」）を日付に変換

## グループチャット対応

家族のLINEグループに招待して使えるようにしました。メンション（`@Bot名`）で呼びかけるとBotが応答します。グループ内の全員のタスクを一元管理できます。

## まとめ

LINE × Claude API の組み合わせは、**日常的に使われるAIアシスタント**を作るのに最適です。高度なNLU能力により、「あれやっといて」のような曖昧な指示も理解してくれます。家族だけでなく、小規模チームのタスク管理にも応用できます。
